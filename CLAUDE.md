## 2. БИЗНЕС

**Тип:** B2B, услуги + поставка оборудования
**Ниша:** Поставка, монтаж, обслуживание лифтового оборудования Sanyo, строительство шахт
**Ключевой партнёр:** Sanyo Elevator (эксклюзивный/приоритетный фокус)

**Услуги:**
- Поставка лифтов Sanyo (пассажирские, грузовые, больничные, панорамные, авто)
- Монтаж (собственные бригады)
- Техническое обслуживание
- Строительство шахт (полный цикл)
- Модернизация
- Проектирование

**Аудитория:** застройщики, коммерческая недвижимость (торговые центры, БЦ, МФК)
**Средний чек:** 1–5 млн руб.
**География:** Москва и МО, потенциально РФ

**Конверсия:** заявка на КП или звонок

## 4. SEO-СТРАТЕГИЯ

**Поисковик-приоритет:** Яндекс (первичный), Google (вторичный)
**Подход:** Яндекс-first: Host в robots.txt, XML-карта для Яндекса, микроразметка Schema.org

**Приоритетные страницы (в порядке SEO-ценности):**
1. `/nonstandard/` — нестандартные шахты (золотая семантика, низкая конкуренция)
2. `/` — главная (брендовые + общие запросы)
3. `/equipment/passenger/` — пассажирские лифты
4. `/services/maintenance/` — техобслуживание лифтов
5. `/services/installation/` — монтаж лифтов

**Семантические кластеры:**
- "лифты sanyo москва" / "купить лифт sanyo"
- "монтаж лифтов в нестандартных шахтах"
- "обслуживание лифтов застройщики"
- "лифт для торгового центра"
- "больничный лифт москва"

**Конкуренты:**
| Конкурент | Слабость |
|---|---|
| Meteor Lift (ex-Otis) | Дорого, негибко, бюрократия |
| Корона-Лифт | Мультибренд без фокуса |
| Союзлифтмонтаж | Массовый сегмент, не B2B |
| ЛифтКомфортСервис | Нет УТП, нет специализации |

**Незанятая ниша:** "инженерная экспертиза + надёжный азиатский бренд без санкционных рисков"

## 6. ДИЗАЙН-СИСТЕМА

**Позиционирование:** premium industrial, инженерная экспертиза, доверие
**Стиль:** тёмный, графитовый фон, золотой акцент — НЕ строительный шаблон

**Цвета:**
```css
--bg-dark:     #0a0a0b      /* основной фон */
--bg-surface:  #111113      /* карточки, секции */
--bg-elevated: #1a1a1d      /* навбар, хедеры */
--text-primary:#f0f0f0      /* основной текст */
--text-secondary:#9898a0   /* второстепенный */
--gold:        #c9a84c      /* акцент */
--border:      #2a2a2e      /* границы */
--border-gold: rgba(201,168,76,0.3)
```

**Типографика:**
- Заголовки: Bebas Neue (display)
- Текст: Golos Text (оптимизирован под кириллицу)

**Принципы:**
- Mobile-first (>60% трафика с мобильных)
- Fade-up анимации при скролле (IntersectionObserver)
- Никаких стоковых паттернов — оригинальный дизайн
